Mode 1 User EXEC Switch>

Giống như "khách" — chỉ xem, không chỉnh được gì. Dùng để kiểm tra tình trạng cơ bản.

Lệnh thường dùng
ping 192.168.1.1Kiểm tra kết nối mạng tới một IP
traceroute 8.8.8.8Xem đường đi của gói tin
show versionXem phiên bản IOS và thông tin phần cứng
logout / exitThoát khỏi thiết bị
Không thể xem cấu hình running-config, không thể đặt IP hay VLAN tại mode này.
Mode 2 Privileged EXEC Switch#

Giống như "admin đọc toàn quyền" — xem mọi thứ, chạy lệnh chẩn đoán nâng cao, lưu/xóa cấu hình.

Xem & chẩn đoán
show running-configXem toàn bộ cấu hình đang chạy
show ip interface briefXem trạng thái tất cả interface (up/down, IP)
show vlan briefXem danh sách VLAN và port gán
show mac address-tableXem bảng MAC — debug kết nối thiết bị
show interfaces Fa0/1Xem chi tiết 1 interface: lỗi, tốc độ, duplex
Lưu & quản lý cấu hình
copy running-config startup-configLưu cấu hình (bắt buộc sau khi config xong)
copy running-config tftpBackup cấu hình ra server TFTP
erase startup-configXóa sạch cấu hình — reset về mặc định
reloadKhởi động lại thiết bị
debug ip ospf eventsBật debug real-time để troubleshoot giao thức
Mode 3 Global Configuration Switch(config)#

Nơi cấu hình toàn hệ thống — tên thiết bị, bảo mật, routing, VLAN, banner đăng nhập.

hostname Core-SW1Đặt tên thiết bị
enable secret Cisco@123Đặt mật khẩu enable (mã hóa MD5)
service password-encryptionMã hóa tất cả mật khẩu trong config
ip routingBật tính năng routing (Layer 3 switch)
vlan 10Tạo VLAN 10 (rồi đặt tên bên trong)
banner motd # Canh bao! #Hiển thị thông báo khi ai đó login vào
no ip domain-lookupTắt tra cứu DNS — tránh delay khi gõ nhầm
Mode 4 Sub-configuration (config-if)# / (config-line)# / (config-router)#

Cấu hình chi tiết từng thành phần: cổng mạng, đường truy cập (console/telnet/SSH), giao thức định tuyến.

Interface — cấu hình cổng vật lý / VLAN SVI
interface FastEthernet0/1Vào cổng Fa0/1
ip address 192.168.1.1 255.255.255.0Gán IP cho interface (Layer 3)
switchport mode access / trunkChế độ access (1 VLAN) hoặc trunk (nhiều VLAN)
switchport access vlan 10Gán cổng vào VLAN 10
no shutdownBật cổng (mặc định cổng bị shutdown)
duplex full / speed 100Cấu hình tốc độ và duplex thủ công
Line — bảo mật đường truy cập
line console 0Vào cấu hình cổng console (dây cắm trực tiếp)
line vty 0 4Vào cấu hình Telnet/SSH (5 session đồng thời)
password cisco123 / loginĐặt mật khẩu cho đường vty/console
transport input sshChỉ cho phép SSH, chặn Telnet (plain text)
exec-timeout 5 0Tự logout sau 5 phút không hoạt động
Router — cấu hình định tuyến
router ospf 1Bật OSPF process 1
network 10.0.0.0 0.0.0.255 area 0Khai báo mạng tham gia OSPF
router rip / version 2Bật giao thức định tuyến RIP v2
